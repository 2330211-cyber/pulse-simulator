Pulse Modulation Simulator

A Web-Based Demonstration of PAM, PWM & PPM

▶ Open Simulator

Abstract

This project demonstrates Pulse Amplitude Modulation (PAM), Pulse Width Modulation (PWM) and Pulse Position Modulation (PPM) using an interactive web-based simulator. Users can vary signal amplitude, sampling frequency and waveform type. The tool also visualizes demodulation to help students understand the reconstruction of analog signals from pulse-modulated forms. This simulator provides a modern, browser-powered alternative to hardware communication trainer kits.

Aim

To study and visualize Pulse Amplitude Modulation (PAM), Pulse Width Modulation (PWM) and Pulse Position Modulation (PPM) using a custom-built web-based simulation tool.

Features

  • Visualizes PAM, PWM and PPM in real time.
  • Adjustable amplitude, waveform type, analog frequency and sampling frequency.
  • Demodulation mode to view recovered waveform.
  • Supports sine, square and triangle input waveforms.
  • Fully browser-based — works on phone, laptop and tablet.
