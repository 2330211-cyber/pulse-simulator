üìù Abstract

This project demonstrates Pulse Amplitude Modulation (PAM), Pulse Width Modulation (PWM) and Pulse Position Modulation (PPM) using an interactive web-based simulator. Users can vary signal amplitude, sampling frequency, and waveform type. The tool also visualizes demodulation to help students understand the reconstruction of analog signals from pulse-modulated forms. This simulator provides a modern, browser-powered alternative to traditional hardware communication trainer kits.
