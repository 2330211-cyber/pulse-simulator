üìù Abstract

This project demonstrates the working principles of Pulse Amplitude Modulation (PAM), Pulse Width Modulation (PWM), and Pulse Position Modulation (PPM) using a web-based interactive simulator. The tool dynamically generates input signals, applies modulation, and visualizes the resulting waveforms along with optional demodulation. By adjusting signal parameters in real-time, users gain a deeper understanding of sampling theory, pulse shaping, and low-pass filtering. The simulator provides an accessible platform for Communication Systems learning in the B.Sc. Physics curriculum.
